## 1 介绍

### 1.1 官方介绍

Sanic 是 Python3.7+ Web 服务器和 Web 框架，旨在提高性能。它允许使用 Python3.5 中添加的 `async`/`await` 语法，这使得您的代码有效的避免阻塞从而达到提升响应速度的目的。

 Sanic 不仅仅是一个 **框架**，它还是一个 **Web 服务器**。

### 1.2 特征

- 内置极速 web server
- 生产准备就绪
- 极高的拓展性
- 支持 ASGI
- 简单直观的 API 设计
- 社区保障
